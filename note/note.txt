✅.gitattributes 파일 생성 위치와 방법
$ git rev-parse --show-toplevel
$ touch .gitattributes

# Force LF line endings for Linux / PetaLinux
* text eol=lf

# Shell scripts
*.sh text eol=lf

# PetaLinux config / project files
*.project text eol=lf
*.config text eol=lf

$ git status
$ git add .gitattributes
$ git commit -m "Add .gitattributes to enforce LF line endings"
